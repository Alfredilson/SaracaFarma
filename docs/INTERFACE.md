# Interface Gráfica

## Tela de Login
- Campos: Usuário, Senha
- Botão: Entrar
- Mensagens: sucesso ou erro

## Tela Principal
- Botões: Cadastro de Medicamentos, Controle de Estoque, Relatórios, Funções Administrativas
- Barra superior com título

## Tela Administrativa
- Botões: Cadastrar novo usuário, Alterar dados do admin
